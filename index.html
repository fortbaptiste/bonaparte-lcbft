01 FONCTIONNEMENT DU SYSTEME

Comment ca marche ?

C'est tout simple. Tu n'as rien a faire. Le systeme tourne tout seul, toutes les 5 minutes. Voici le pipeline complet, de la signature au depot du rapport.

1

Un contrat est signe sur MyNotary

Des qu'un mandat de vente, un mandat de recherche ou une offre d'achat est signe, le systeme le detecte automatiquement. Le webhook MyNotary envoie l'evenement en temps reel.

Mandat de vente Mandat de recherche Offre d'achat
2

Verification Efficiale

Le systeme verifie chaque personne sur les listes officielles : Gel des avoirs (sanctions internationales) et PPE (personnes politiquement exposees). Pour les societes, on verifie aussi les beneficiaires effectifs via l'API Pappers.

API calls: efficiale.verify() + pappers.beneficiaires()
3

Le rapport PDF est genere

Un rapport de conformite est cree pour chaque personne verifiee, avec tous les details : identite complete, resultat des recherches sur chaque liste, alertes eventuelles avec le detail des correspondances, et la date de verification.

4

Depot automatique dans MyNotary

Le rapport est uploade dans l'espace de stockage du dossier MyNotary correspondant. Tu le retrouves directement dans le dossier, sans rien faire. Le systeme gere aussi le nommage et le classement.
